# TP 30 – Pipeline CI/CD avec Jenkins, GitHub, Docker et ngrok

## Description du projet

Ce projet met en place un pipeline CI/CD complet pour une application Spring Boot, en utilisant Jenkins, GitHub, Docker et ngrok.
L’objectif est d’automatiser les étapes de build, test, conteneurisation et déploiement, avec un déclenchement automatique à chaque push GitHub.

## Technologies utilisées

- Java 17

- Spring Boot

- Maven

- Jenkins

- Docker & Docker Desktop

- Git & GitHub

- ngrok
